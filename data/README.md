# EMOTIC Dataset

## Download Instructions

1. Visit the official EMOTIC website
2. Request dataset access
3. Download and extract to this folder

**Dataset structure:**
data/
├── emotic/
│   ├── ade20k/
│   ├── emodb_small/
│   ├── framesdb/
│   └── mscoco/
└── Annotations/
    └── Annotations.mat

**Note:** Dataset files are not included in this repository due to size constraints.
